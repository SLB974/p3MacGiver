Game "Hep MacGyver to Escape"

Rules :     MacGyver must quit the maze after having picked up three items : Needle, Ether, Tub.
            He can quit the maze going to Guardian position.
            If he has picked all the items, he wins else he dies and the game is over.

            MacGyver and guardin positions are defined in the maze structure file.
            Items' positions are randomly defined at start.

            MacGyver is moved by keyboard direction's keys.

Informations :

            Items' list is written in constants.py file.
            So we can change number of items anytime.
